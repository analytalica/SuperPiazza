Version 1.0.0 (Master branch)

SuperPiazza is a styling overhaul for Piazza that recolors, rearranges, and resizes page elements for a more streamlined and modern usage experience. Newly added contrasts in color enable users to navigate Piazza more efficiently and improve its readability.
